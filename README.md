### File Management System

### Number Guessing Game

  📁 Folder: `Number_Guessing`

### QR Code Generator

  📁 Folder: `QR_Generator`

## How to Run
1. Open terminal or command prompt
2. Navigate to the project folder:

   ```bash
   cd folder_name
   ```
3. Run the Python file:

   ```bash
   python filename.py
   ```
